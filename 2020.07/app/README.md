## Задание

1. Получить данные из файла data.json и вывести их в рублях (курс выбрать произвольно) на страницу
2. Сопоставления id групп и товаров с их названиями находятся в файле names.json
3. После вывода данных навесить обработчики для добавления выбранного товара в корзину и удаления из нее
4. Сделать рассчет общей суммы товаров и вывести отдельным полем
5. Корзина находится на одной и той же странице вместе со списком товаров
6. Вывести данные используя привязку к представлению и возможностью последующего изменения (two-way binding). Можно использовать фреймворки.
7. Сделать обновление цены товара в зависимости от курса валюты
8. С интервалом в 15 секунд читать исходный файл data.json и одновременно менять курс доллара (вручную) на значение от 20 до 80, выполняя обновление данных в модели (с изменением в представлении). Если цена увеличилось в большую сторону - подсветить ячейку красным, если в меньшую - зеленым.

## Навигация по проекту

- В папке assets находятся файлы data.json и names.json
- В папке core находятся хелперы для решения общих задач
- В папке shared находятся данные для всех компонентов

## О проекте

Проект состоит из двух основных частей:

- Каталог
- Корзина

Через элемент управления можно поменять курс доллара в диапазоне от 20 до 80.
Раз в 15 секунд происходит обновление данных в каталоге, изменяя цены товара по новому курсу.
Цены в корзине зафиксированы, так как добавляя товар в корзину пользователь соглашается на текущую цену (так же в тестовом задании по этому вопросу ничего не сказано).
