# App

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.8.

## Запуск проекта

- npm i
- npm run server

## Навигация по проекту

- В папке assets находятся файлы data.json и names.json
- В папке core находятся хелперы для решения общих задач
- В папке shared находятся данные для всех компонентов

## О проекте

Проект состоит из двух основных частей:

- Каталог
- Корзина

Через элемент управления можно поменять курс доллара в диапазоне от 20 до 80.
Раз в 15 секунд происходит обновление данных в каталоге, изменяя цены товара по новому курсу.
Цены в корзине зафиксированы, так как добавляя товар в корзину пользователь соглашается на текущую цену (так же в тестовом задании по этому вопросу ничего не сказано).
