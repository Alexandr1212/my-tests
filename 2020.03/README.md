# Тестовое задание 03.2020

## О проекте

1. backend написан на Node.js
2. frontend написан на JavaScript

## Используемые технологии

1. HTML, CSS
2. JavaScript, Node.js

## Запуск проекта

1. запускаем сервер: node server.js
2. открываем http://localhost:3000

## Задание

1. Инструменты: HTML, CSS, JavaScript. Без использования библиотек, фреймворков.
2. Страница состоит из трех частей: фильтр, график, таблица.
3. По умолчанию пользователь видит последние 7 записей.
4. Данные хранятся в файле metrics.json.
5. После прочтения данные должны попасть в IndexedDB.
6. Для построения таблицы и графика данные берутся из IndexedDB.
7. В фильтре доступны даты, которые есть в файле metrics.json.
8. Выбор метрики на графике (все кроме даты).
9. Сортировка таблицы по колонкам.
