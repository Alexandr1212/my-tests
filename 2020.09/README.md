# App

Тестовое задание на вакансию "Верстальщик".

## Запуск Gulp

- yarn install
- gulp

## Gulp

- Компилирует scss в css, минифицирует, добавляет префиксы
- Собирает все js скрипты в одном файле и минифицирует его
- Сжимает изображения

## Навигация по проекту

- В папке src находятся изображения, js скрипты, scss файлы. Все они будут обработаны gulp
- В папке dist находятся файлы для корректной работы сайта в браузере ie и файлы которые прошли gulp
